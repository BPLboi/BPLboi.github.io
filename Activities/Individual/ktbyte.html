KTBYTE

Overview

KTBYTE is a computer science academy based in Lexington MA, the town where I grew up. There, I was introduced to robotics, learned to code, and eventually became a teacher for some of their classes during middle and high school.

Robotics Teacher

During high school, I worked as a robotics instructor that taught young students how to use the Arduino microcontroller. To do this, I created materials explaining how to use various types of sensors, and also taught students how to code.

Teaching Assistant

Throughout high school, I worked as a teaching assistant for KTBYTE office hours. There, I helped students with the coding classes they were taking, which usually involved answering questions about inheritance, polymorphism, recursion, and data structures.

Coding Classes

Throughout middle school (and part of high school), I took (mostly) Java-based coding classes at KTBYTE to learn how to code! Here are some of the classes I took:

  • CS00a -- Intro to Computer Science with Java
  • CS00b -- Computer Science and Applications
  • CS01a -- Algorithms and Problem Solving in Java
  • CS02a -- Intermediate Algorithms
  • CS02b -- Applied Data Structures
  • USACO Silver
  • USACO Gold
  • Intro to Machine Learning

Through the CS0xx classes, I learned how to program in Java, and covered loops, conditions, recursion, inheritance, polymorphism, and commonly used data structures (Stacks, Queues, Hash/Tree maps and sets, etc.). Towards the end of the course series, I did a project constructing Huffman Trees to compress text, and then analyzing how effective the compression was.

The USACO classes taught me about common computer science algorithms and tricks used to speed up computation, such as memoization, graph algorithms (breadth-first, depth-first, and Djikstra's), prefix sums, sliding windows, and more.

In the Machine Learning class I took, I learned how to use (with libraries) a few different machine learning models and environments to analyze datasets found on Kaggle.

Robotics Club

After the end of 5th grade, I signed up for a class at the KTBYTE Robotics club. There, I learned to use Arduino-s and sensors to create my own projects. The most notable project I made is a Firefighter Robot.
